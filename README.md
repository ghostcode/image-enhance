# 图片美化工具

一款基于 HTML5 Canvas 的在线图片美化工具，无需后端服务，纯前端实现所有图像处理算法。界面采用深色专业编辑器风格，支持实时预览与高清导出。

## 功能特性

### 基础调整
- **亮度**：调整图像整体明暗程度
- **对比度**：控制画面亮部与暗部的差异
- **饱和度**：调节色彩鲜艳程度
- **色温**：冷暖色调偏移调整
- **曝光**：模拟相机曝光补偿效果
- **锐化**：增强图像边缘细节

### 预设滤镜
内置 10 种专业风格滤镜，一键应用：
- 原图、复古、黑白、暖阳、冷调
- 戏剧、褪色、鲜艳、电影、怀旧

### 图像变换
- **左旋转**：逆时针旋转 90°
- **右旋转**：顺时针旋转 90°
- **水平翻**：水平镜像翻转

### 其他功能
- 拖拽或点击上传图片
- 实时预览所有调整效果
- 保持原始分辨率导出 PNG 格式
- 一键重置所有参数

## 技术实现

- **纯原生技术栈**：HTML5 + CSS3 + Vanilla JavaScript
- **Canvas 像素级处理**：通过 `getImageData` / `putImageData` 实现像素级操作
- **无依赖**：无需任何第三方库或框架
- **响应式布局**：适配桌面端与移动端

## 使用方法

1. 下载 `index.html` 文件
2. 使用现代浏览器直接打开
3. 上传图片开始编辑
4. 调整参数或选择滤镜
5. 点击「下载」保存处理后的图片

## 核心算法

### 亮度调整
```
R' = R + brightness * 255
G' = G + brightness * 255
B' = B + brightness * 255
```

### 对比度调整
```
factor = (contrast + 100) / 100
R' = ((R/255 - 0.5) * factor + 0.5) * 255
```

### 饱和度调整
```
gray = 0.2989*R + 0.587*G + 0.114*B
R' = gray + saturation * (R - gray)
```

### 锐化处理
采用拉普拉斯锐化卷积核：
```
[ 0, -1,  0]
[-1,  5, -1]
[ 0, -1,  0]
```
